Allow only one sticky post at a time.

== Description ==

When saving a post that has been flagged as sticky, it automatically unsticks other posts so that there's only
one sticky post a a time.

== Installation ==

1. Upload the `lonely_sticky` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
